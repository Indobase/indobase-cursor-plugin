Help me authenticate Cursor with Indobase.

Use this flow:

1. Confirm that the `indobase` MCP server is installed.
2. If the MCP server is not authenticated, tell me to open **Settings > Cursor Settings > Tools & MCP** and authenticate the `indobase` server.
3. After authentication, verify access by using MCP account tools to list organizations or projects.
4. If a project is needed, help me choose one and continue from there.
5. Prefer the hosted MCP login flow over asking me for a manual access token.
